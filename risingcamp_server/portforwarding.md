# 리눅스 가상호스트
리눅스는 가상호스트를 두어 같은 ip 내에서 다른 uri를 요청받고 그에따른 다른 응답을 줄 수 있다.   
이를위해 서버블록을 아래와 같은 설정한다. 
```
server {
    server 도메인 (다른 uri)
    root 응답할 페이지가 있는 디렉토리
    그에따른 원하는 행동
}
```

# 리눅스 문법 조금
chmod : 리눅스는 디렉토리마다 사용자와 그룹을 둔다. change mode 로써 해당 권한설정을 변경 할 수 있다.
  
ln -s : 심볼릭 링크로 디렉토리 바로가기를 설정할 수 있다.
